# DeHassle 🚀

**DeHassle** is a beautifully crafted, gamified, and highly secure productivity engine. Designed for professionals and students alike, it transforms standard task management into an engaging, data-rich experience that actively encourages deep work and consistent habits.

![DeHassle Dashboard Demo](https://via.placeholder.com/1200x600.png?text=DeHassle+Productivity+Dashboard) *(Replace with a screenshot of your dashboard)*

## ✨ Key Features

* **🎮 RPG Gamification & Streaks:** Turn productivity into a game. Earn XP for completing tasks and running focus timers. Level up from *Novice Planner* to *Grandmaster* and maintain daily streaks to build lasting habits.
* **📊 Executive Dashboard:** Visualize your workflow with a 30-day GitHub-style activity heatmap, dynamic task-distribution pie charts, and an "Overdue Action Center" to catch slipped tasks.
* **🧠 Smart Insights:** The app analyzes your completion timestamps to calculate your peak productivity hours and week-over-week task velocity.
* **🗓️ Interactive Calendar View:** A full-scale calendar UI. Drag and drop unscheduled tasks directly onto dates to effortlessly plan your week.
* **🍅 Integrated Focus Timer:** A built-in Pomodoro timer inside the task details. Letting the timer hit zero awards you with a "Focus Spark" (⚡), forever immortalizing your deep work on that task.
* **🔐 Enterprise-Grade Security (2FA):** Built-in Two-Factor Authentication (TOTP). Secure your account with authenticator apps like Google Authenticator or Authy.
* **⌨️ Lightning-Fast Shortcuts:** Fully keyboard accessible. Press `?` anywhere to pull up the global command menu for rapid navigation and task creation.
* **🌙 Sleek, Responsive UI:** A premium, fluid interface powered by Tailwind CSS that looks stunning on massive desktop monitors and perfectly optimized for mobile devices.

## 🛠 Tech Stack

* **Frontend:** [React](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Backend & Auth:** [Supabase](https://supabase.com/) (PostgreSQL, Row Level Security, Multi-Factor Authentication)
* **Icons:** [Lucide React](https://lucide.dev/)
